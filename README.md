# 🏆 Vendor Performance Scorecard

## Use Case
Logistics network had no structured way to evaluate vendors — reviews were periodic, manual, and opinion-driven. Leadership needed a consistent, data-backed view of which vendors deserve more business and which need intervention.

## How It Helped
Built a live scoring engine across 10 KPIs that automatically tiers vendors (A–D), flags underperformers, proposes lane-level SOB reallocation, and quantifies cost savings — replacing gut-feel with a governed monthly framework.

## My Role
Designed the scoring logic, built the full dashboard, set up the monthly review cadence, and circulated outputs as the official vendor performance rating.

## Views

**Category View** — Vendors auto-tiered (A–D) by monthly trip volume; shows network concentration and top contributors at a glance.
![Category View](page1_category_view.png)

**Scorecard Matrix** — Every vendor scored across 10 weighted KPIs; performance gaps visible instantly with top/bottom performers highlighted.
![Scorecard Matrix](page2_detailed_matrix.png)

**Lane Insights** — SOB proposals, cost savings engine, and MoM consistency tracker per vendor per lane with Increase / Maintain / Decrease recommendations.
![Lane Insights](page4_lane_insights.png)

**Commercials** — Vendor rates benchmarked against L1 (lowest) rate per lane; identifies overpriced and underperforming vendors simultaneously.
![Commercials](page5_commercials.png)

## Output
5-page live dashboard refreshed monthly — vendor scores, rank trends, SOB proposals, cost savings engine, and rate vs benchmark analysis. Formally reviewed at leadership level.
